# Handwritten Digit Recognition

A deep learning project that recognises handwritten digits (0–9) using a custom CNN trained on the MNIST dataset, with an interactive Streamlit web app for real-time predictions.

---

## Demo

Upload a photo of a handwritten digit and the model returns the predicted digit along with confidence scores for all 10 classes.

---

## Project Structure

```
├── app.py                          # Streamlit web application
├── mnist_digit_recognition.ipynb   # Model training notebook
├── mnist_cnn.h5                    # Trained model weights (generated by notebook)
├── mnist_train.csv                 # MNIST training data (60,000 samples)
├── mnist_test.csv                  # MNIST test data (10,000 samples)
├── confusion_matrix.png            # Confusion matrix from training
├── training_history.png            # Training/validation accuracy & loss curves
├── Test Numbers/                   # Sample test images
├── requirements.txt
└── README.md
```

## Model

| Detail | Value |
|---|---|
| Architecture | 3-block Deep CNN |
| Input | 28 × 28 grayscale |
| Output | 10-class softmax |
| Training samples | 60,000 |
| Test samples | 10,000 |
| Max epochs | 50 (early stopping) |
| Test accuracy | 99.4 – 99.6% |
| Framework | TensorFlow 2.21.0 |

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/handwritten-digit-recognition.git
cd handwritten-digit-recognition
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Add the data files

Place `mnist_train.csv` and `mnist_test.csv` in the project root. These can be downloaded from [Kaggle – Digit Recognizer](https://www.kaggle.com/competitions/digit-recognizer/data).

### 4. Train the model

Open and run all cells in the Jupyter notebook:

```bash
jupyter notebook mnist_digit_recognition.ipynb
```

This will produce `mnist_cnn.h5` in the project root.

### 5. Launch the app

```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

## Using the App

**Upload Image tab**
- Upload a PNG, JPG, or BMP image containing a single handwritten digit.
- The app preprocesses it to MNIST format (28 × 28 grayscale, white digit on black background) and displays the original, the processed input, and the prediction with confidence.

**Random Test tab**
- Loads a random sample from `mnist_test.csv` and predicts it live, showing whether the model is correct.

### Tips for best accuracy
- One digit per image
- Dark ink on a white or light background
- Thick, clear strokes
- Digit fills most of the frame
- No shadows or background patterns
- Photograph straight-on, not at an angle
